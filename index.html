HTML

  • HTML is hypertext markup language and gives content structure
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Meta data is data about a webpage that the browser uses to effectively serve webpages to users

CSS

  • CSS is a programming language used by developers to define how a webpage content is presented to users
  • When a developer wants to style an element directly in the HTML file, they use inline CSS
  • External CSS stylesheet allows developers to keep all their CSS rules in a seperate files, which makes design change easier

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch-name:creates a new branch and switches to it
  • We use git in our local environment to enable version control
  • In the feature branch, we make commits of all the changes we make to the code while working
  • A pull request is used to merge a comparison branch/feature branch into the main/base branch
